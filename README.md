ASIN番号からJAN(EAN)番号を取得するPythonスクリプト
==============================================

### 実行手順
* % python asin2jan.py <ASIN番号>  # JANコードを表示する

### 実行環境
* Mac OS X (10.6.8)
* Python (2.6.7)
    * 外部依存モジュール
        * argparse module
        * Pit module

### 使用WebAPI
* Amazon(AWS) Product Advertising API
